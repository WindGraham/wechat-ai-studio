# 编辑器功能清单

## 一、核心定位

一个纯静态网页编辑器，用于和 AI 协作排版微信公众号文章。用户拖拽调整布局、精细调控样式，AI 通过内置终端（xterm.js + WebSocket）参与协作。

## 二、交互模式

### 2.1 两种布局模式

| 模式 | 行为 | 输出方式 |
|:---|:---|:---|
| **流式模式** | inline-block 垂直堆叠，拖拽调顺序 | 直接导出文档流 HTML |
| **自由模式** | 绝对坐标拖拽，层叠错落 | 坐标换算为负 margin + padding |

两个模式共存，每个块独立切换，可随时在两个模式之间切换。

### 2.2 组件来源

- 从 GitHub Pages ModeHub 加载 `catalog.json`
- 左侧面板展示缩略图 + 编号 + 名称（分类折叠）
- 插入方式：点击插入到当前位置，或拖入到画布指定位置
- 模板（整页）和组件（零件）不严格区分，统一处理
- 编辑器自身也可插入 GitHub Pages 或任意 URL 的组件

### 2.3 用户操作

| 操作 | 方式 |
|:---|:---|
| **拖拽块** | 自由模式：任意方向；流式模式：限制 Y 轴 |
| **缩放块** | 8 方向拖拽角/边 |
| **精调样式** | 右侧/悬浮属性面板（VvvebJs inputs.js 移植） |
| **删除块** | 键盘 Delete 或工具栏删除按钮（不用右键，避免浏览器菜单冲突） |
| **新增块** | 左侧面板点击/拖入，或画布空白处快捷插入 |
| **合并组** | 框选多个 section → 工具栏"合并为组" |
| **解组** | 选中组 → 工具栏"解散组" |
| **图片** | 拖入本地文件或粘贴公网 URL |
| **撤销/重做** | Ctrl+Z / Ctrl+Shift+Z（DOM 快照方案） |

### 2.4 属性面板

选中 block 后，面板展示对应控件：

| 类型 | 推断依据 | 显示控件 |
|:---|:---|:---|
| 通用（全部适用） | 总是显示 | margin/padding 四向、背景色、透明度 |
| 标题区 | font-size≥20px 且含 `<strong>` | 字号、颜色、对齐、背景色 |
| 正文区 | 含 `text-indent:2em` | 字号、颜色、行高、对齐、缩进 |
| 卡片 | 含 background-color+border/box-shadow | 字号、颜色、圆角、边框、阴影、背景色 |
| 图片 | 含 `<img>` | 替换 URL、裁剪参数 |
| 分割线 | 高度<10px 或纯装饰 | 宽度、颜色、线型 |

类型根据元素 style 属性自动推断，不需要 AI 额外标记。

### 2.5 文件与版本

- 每个文档在工作目录下管理 `article.html`
- 用户点"保存"时：画布内容导出 → 覆盖 `article.html` → 刷新画布
- 每次改动生成 git commit（用户点保存或手动触发）
- 保留记录，用户需要 AI 修改时，AI 读取 git 历史获取上下文

## 三、AI 协作

### 3.1 终端集成

- 使用 **xterm.js** 在前端渲染终端
- 本地跑 **Python WebSocket 后端** (localhost:8090)
- 后端启动 opencode 进程，stdin/stdout 双向透传
- 不同标签页各自对应独立终端（实际上用户只用一个）

### 3.2 协作流程

```
用户 → 编辑器拖拽/精调 → 保存 article.html → 
  终端告诉 AI "我改了布局，看 git diff" → 
  AI 分析修改 → 生成新 HTML → 
  覆盖 article.html → 用户点保存 → 编辑器刷新
```

### 3.3 模板/组件选择

- 用户在 ModeHub Pages 浏览模板
- 记住模板编号，在终端告诉 AI "用 #0421 模板"
- AI 从 ModeHub 取模板 HTML 写入 article.html
- 编辑器加载显示，用户继续调整

## 四、技术选型

| 层 | 库 | 用途 |
|:---|:---|:---|
| 拖拽引擎 | Interact.js | 自由拖动、8 向缩放、吸附对齐 |
| 画布渲染 | 原生 DOM | section 就是 HTML，所见即所得 |
| 属性面板 | VvvebJs inputs.js | 字号/颜色/间距/圆角/阴影控件 UI |
| 终端 | xterm.js + WebSocket | 浏览器内嵌终端，转发 opencode |
| 后端 | Python asyncio/websockets | WebSocket 服务，管理 opencode 进程 |
| ModeHub | GitHub Pages + 自建页面 | 模板/组件浏览，按需取 HTML |
