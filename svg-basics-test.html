SVG BASICS TEST

SVG 基础功能检测

只测确认可用的核心功能

测试目的

本文仅包含微信编辑器中确认可用的 SVG 基础功能。若某项显示为空白或效果异常，说明该基础功能在微信中实际已被过滤，需调整使用策略。

01 基础形状

BASIC SHAPES

1.1 rect · circle · ellipse · line

预期：红矩形(圆角) / 蓝圆 / 黄椭圆 / 绿直线

1.2 polyline · polygon · path

预期：红色折线 / 蓝色三角 / 绿色贝塞尔曲线 / 黄色闭合路径

02 文字

TEXT & TSPAN

2.1 text · tspan

预期：文字正常显示，中间红蓝 tspan 变色

03 裁剪路径

CLIP PATH

3.1 clipPath

预期：红色矩形被裁剪为圆形

04 滤镜

FILTERS

4.1 feGaussianBlur · 4.2 feComponentTransfer

预期：左蓝圆模糊 / 右红矩形半透明

05 变换属性

TRANSFORM

5.1 translate · scale · rotate · skewX · matrix

预期：五个矩形分别发生位移/缩放/旋转/倾斜/矩阵变形

06 SMIL 属性动画

ATTRIBUTE ANIMATION

6.1 透明度 · 半径 · 颜色 · 描边宽度

预期：四个圆分别发生透明度/半径/颜色/描宽动画

07 SMIL 变换动画

TRANSFORM ANIMATION

7.1 translate · scale · rotate · skewX · matrix

预期：五个矩形分别进行位移/缩放/旋转/倾斜/矩阵动画

08 路径动画

PATH ANIMATION

8.1 stroke-dashoffset 绘制动画

预期：绿色线条从左到右持续绘制

8.2 animateMotion 路径运动

预期：红色圆点沿灰色波浪线运动

09 视口与比例

VIEWBOX & RATIO

9.1 viewBox · preserveAspectRatio

viewBox="0 0 100 50"

slice 裁剪

预期：左栏完整蓝圆 / 右栏红圆被裁剪填充

测试结束

以上为微信编辑器中确认可用的 SVG 基础功能。
若某项显示为空白或效果异常，请立即反馈。
